Types conversion

Convert to a String:

  • By using the global method String().

    Syntax: String(x);

  • By using the Number method toString().

    Syntax: x.toString();

  • By using the ' + ' operator when one of the elements is a string.

    Syntax: '7' + 12; /Returns 712

Convert to a Number:

  • By using the global method Number().

    Syntax: Number(x);

  • By using the function parseFloat().

    Syntax: parseFloat('a string starting with a numeric character');//Returns a floating number.

  • By using the function parseInt().

    Syntax: parseInt('a string starting with a numeric character');//Returns an integer number.

Convert to a Boolean:

  • By using the global method Boolean().

    Syntax: Booleadn(x);

    0, false, null, undefined, '' -> will return false. All the others (everything with a value) will return true.
