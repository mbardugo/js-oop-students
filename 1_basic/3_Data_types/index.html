Data types

JS has two categories of variables: Primitives and Objects.

Primitives

List of the different primitive data types:

  • Number (5, 7.2).
  • String ("5", "Foo", "true", "false", "undefined").
  • Boolean (true, false).
  • Undefined -
    • It is the type of a variable that has been declared but has not been initialized.
    • The type of an undeclared variable.
    • A function that lacks an explicit return statement will return undefined.
  • *Null - It is the value of nothing and it must be assigned to a variable.
  • Symbol (ES6).

Copied by their value.

Are immutable i.e., can't be changed.

Objects

All the listed items are of type Object.

  • *Function (function(){}).
  • Array ([]).
  • Object ({}).


Copied by a reference.

Can be changed.

Use the 'typeof' operator to retrieve the type of an element. Syntax: typeof x;
