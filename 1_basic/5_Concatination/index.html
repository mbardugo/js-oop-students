Concatination using the ' + ' operator

When the operator is between two numbers it will perform an addition.

When at least one of the elements is not a number, the operator will convert the elements on both sidesto strings and concatintate theme.

For concatinaitnig to an existing variable or for increasing its value, use the short syntax '+=''.
