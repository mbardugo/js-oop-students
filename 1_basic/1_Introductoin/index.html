JS Introduction

About JS

  • JS is the browsers programming language.
  • One of the 3 core technologies of the www (Word wide web).
  • It is a very useful tool for data manipulation and the only tool for DOM manipulation.
  • Follows the ECMAScript, standardized by ECMA international (ECMA = European Computer Manufactor Association) .
  • JS by default runs in 'sloppy mode', that means that it 'forgives' incorrect typing and interperates it or fails silenty. For better coding enforcement use strict mode (Sintax: 'use strict') at the beginning of the desired strict scope.

Homework

JS Events

  1. Get yourself familiar with the following JS events:
    • onclick.
    • onsubmit.
    • onkeyup.
  2. Choose at least 2 additional events to learn about.
  3. Write a working example for each of the above.

JS Selectors

  1. Get yourself familiar with the following JS selectors:
    • getElementById().
    • getElementsByClassName().
    • getElementsByTagName().
    • querySelector().
    • nextElementSibling
  2. Write a working example for each of the above.

JS DOM methods and properties.

  1. Get yourself familiar with the following DOM manipulation:
    • innerHTML.
    • createElement().
    • append().
    • appendChild().
  2. Write a working example for each of the above, use innerHTML as a getter and as a setter in the same example.
