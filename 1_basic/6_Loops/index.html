Loops

About

  1. Loops are precious tools for programmers as they gives us the ability to do a repetitive actions for a group of elements by certain criteria.
  2. 'While', 'do while' and 'for' are useful when want to execute code as long as a certain criterion is true. Those loops can iterate through an array using its index.
  3. Iterate through an object with the 'for-in' loop.
  4. Iterate through an array with 'foreach' or with the 'map' methods. The map() method is designed to create a new array and to return it, while the forEach() method returns the value of undefined.

Skip or stop the iteration

  • Break statement - exit the loops mentioned in rows 2-3.
  • Continue statement - skip to the next iteraton of the loops mentioned in rows 2-3.
  • Return statement - acts as the continue statement in the forEach and map methods.
